# Book_Library_BackBone_Responsive_app
Backbone.js is used for this project. This is using goole book apis and it is a responsive app.
TO run this please open chrome with this command : 
 ~ mukteshmohan$ /Applications/Google\ Chrome\ 2.app/Contents/MacOS/Google\ Chrome --allow-file-access-from-files
 
 Change directory "cd Documents/BookLibrary" (to pint to folder where index.html resides)
 
 run the python server to enable CORS.
       ~  "python -m SimpleHTTPServer"
